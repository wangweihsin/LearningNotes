## 何謂演算法?
>把每個值拆成細項步驟，估計花多少時間能夠消化完畢

## 如何debug?
* spyder-debugmode :
  
      Step1. 把print放在想知道的地方             
      
      Step2. 刪掉可能錯的
    
      Step3. 手寫設Breakpoint，看看print出來的跟手寫的一不一樣
   
   `step()` : 直接跳到下一個
   
   `step in()` : 跳到同一個的下一行
   
* visual studio code :
   
   心得 : 可以把游標移到該變數上面，能夠隨著debug的過程中，看到現在變數是如何跑、變化的，個人覺得較spyder好用!
