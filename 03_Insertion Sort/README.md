# insertion sort

    先找到(抓出)要觀察的值往前面和其他所有的值依次比較 
      → 紅色:要插入的項目 黑色:檢查過的值
      → n平方次
      → 轉成流程圖

# 何謂演算法?
    把每個值拆成細項步驟，估計花多少時間能夠消化完畢

# 如何debug?
spyder-debugmode:

    STEP1 把print放在想知道的地方
    STEP2 刪掉可能錯的
    STEP3 手寫設Breakpoint印出來的跟手寫的一不一樣
              step直接跳到下一個
              step in跳到同一個的下一行
