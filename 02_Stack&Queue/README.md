# stack
>堆疊

>>**先進後出**
* 可以想像成「盤子」
  * `push()` : 放盤子到桌上
  * `pop()` : 取出盤子(從最上面取)  
  * `top()` : 告訴使用者最上面長怎樣 但沒被拿走
      
* 可以解謎宮的問題
	
	作法 : STEP1把每一步丟到stack
	       STEP2用push記錄每一步(因為存的時候就具備從最上面取的特性)
	       STEP3故碰壁的時候(要上一步的時候)用一步一步退回去top 
	 
	
# queue
>佇列
* 可以想像成「排隊」

  	第一個到的先出去，不能插隊只能從尾巴差資料
	    isempty:有沒有人在排隊
	    getsize:幾個人在排隊
	
* 排程的需求 
	一次只能解決一個問題 還沒被排到的放在queue，EX:印表機
